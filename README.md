# -healthcare_project-
End-to-end healthcare data analytics project using SQL, Python, Excel, Power BI and Tableau
# Healthcare Revenue & Billing Risk Analysis

This project analyzes 4,999 hospital patient records to identify revenue drivers and billing risks.

## Tools Used
- SQL
- Python (Pandas, NumPy)
- Excel
- Power BI
- Tableau

## What was done
- Cleaned and standardized healthcare data
- Identified negative billing and revenue leakage
- Created KPIs and dashboards
- Built Tableau story for business insights

## Files
- healthcare_cleaned.csv – cleaned dataset
- analysis.py – Python data cleaning and analysis
- healthcare_sql_analysis.sql – SQL queries
- Power BI and Tableau dashboards
